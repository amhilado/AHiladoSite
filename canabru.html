Canabru Website Design and Development

Goal: Design and develop a responsive website for a local coffee shop.

Project Focus: User Interface Design and Front End Development

Software Used: Adobe XD, Adobe Photoshop, Visual Studio Code, GitHub, GitKraken

Final Deliverables: Fully responsive website

Design Thinking

Canabru is a small local coffee shop located in Chino, CA. They currently do not have a working website, so I decided that I would take the time to design one for them as a concept project.

The main aspect I wanted to focus on for this website was usability, ease of navigation, and a visually appealling user interface.

The live website can be viewed at: https://amhilado.github.io/Canabru/
